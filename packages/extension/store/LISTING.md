## Name
GitBaz - GitHub Contributor Context

## Short Description (132 chars max)
Sharp as a falcon - see contributor scores, PR stats, badges, bus factor, and activity heatmaps on any GitHub page.

## Detailed Description

GitBaz adds contributor context directly to GitHub pages. See who you're working with at a glance - no tab-switching needed.

"Baz" means "falcon" in Kurdish - the animal with the sharpest eyesight.

FEATURES:

• Contributor Scores - 100-point scoring across PRs, commits, merge rate, account age, and followers with tier badges (Newcomer -> Maintainer).

• Smart Badges - First-Timer, Issue Reporter, Sharp Eye, Prolific, Veteran, Popular, New Account, and more.

• PR & Issue Context - See contributor history and reputation directly in PR and issue sidebars.

• Discussion Panels - Context for GitHub Discussions and org-level discussions.

• Bus Factor Analysis - Identify single-contributor risk at the repository level.

• Knowledge Silos - Detect file ownership concentration via git blame on PRs.

• Activity Heatmaps - Visualize contributor activity patterns and streaks.

• Bot Detection - Automatically identify bot accounts.

• AI-Generated PR Detection - Flag PRs that appear to be AI-generated.

• Repository Health - OpenSSF Scorecard integration for security posture.

• Vouch System - Community-driven vouching via VOUCHED.td files.

PRIVACY:

All data stays in your browser. Your GitHub PAT is stored locally and never sent anywhere except the GitHub API. No analytics, no tracking, no external servers. Fully open source.

SETUP:

1. Install the extension
2. Click the GitBaz icon -> Options
3. Enter your GitHub Personal Access Token
4. Navigate to any GitHub issue, PR, or discussion

Source code: https://github.com/happyhackingspace/gitbaz
